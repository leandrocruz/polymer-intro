Polymer

Why polymer ?

Polymer is a framework build on top of Web Components that provides an aditional set of custom components and extra functionality like two way data binding.

Polymer

Seriously, why?

  1. Mobile and offline support
  2. Highly reusable
  3. Declarative approach
  4. Large set of pre-built components
  5. Good documentation/community
  6. Enables good practices
  7. Modern pipeline and tooling
  8. Material Design
  9. Themeable
